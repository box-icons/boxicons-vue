# Changesets

Hello! This is where we keep track of changes to this package.

## Adding a changeset

To add a changeset, run `npx changeset` in the root of this repository. Follow the prompts to select the packages you've changed and specify the version bump (patch, minor, or major).

## Versioning and Publishing

When you're ready to release, run `npx changeset version` to bump versions and update changelogs. Then run `npm publish` to release the packages.
